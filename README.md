# Kenya Inflation Drivers Analysis

An end-to-end data analysis project exploring the key drivers of inflation in Kenya from 2000 to 2024.

## Tools
Python, pandas, matplotlib, seaborn, statsmodels, Streamlit

## Project Structure
- data/ — raw and cleaned datasets
- notebooks/ — exploratory analysis
- src/ — cleaning and modelling scripts
- dashboard/ — interactive Streamlit app
